# Bus Tracking Driver/Admin System

An Android application developed for drivers and administrators to manage and update live college bus locations using Firebase and GPS.

---

##  Features

- Driver Login System
- Live GPS Bus Location Updates
- Real-Time Firebase Sync
- Google Maps Integration
- Route Management
- Bus Information Management
- Student Tracking Support
- SOS Monitoring Support

---

## 🛠 Technologies Used

- Android Studio
- Kotlin
- XML
- Firebase Authentication
- Firebase Realtime Database
- Google Maps API

---

##  Modules

### Driver Dashboard
- Start bus tracking
- Update live location
- View assigned route
- Manage bus status

### Admin Features
- Manage buses
- Manage routes
- Monitor tracking system

### Live Tracking
- Sends real-time GPS updates
- Syncs location to Firebase
